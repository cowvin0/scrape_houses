Instalar dependências: 
    poetry install 

Entrar no ambiente: 
    poetry shell 

Iniciar código de raspagem: 
    make scrape city=nome_cidade 

Padrões de nome de cidades do ZapImóveis: 
    João Pessoa: pb+joao-pessoa 
    Boa Vista: rr+boa-vista 
    Maricá: rj+marica 
    Recife: pe+recife
