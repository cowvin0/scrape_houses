Instalar dependências: \n
    poetry install \n

Entrar no ambiente: \n
    poetry shell \n

Iniciar código de raspagem: \n
    make scrape city=nome_cidade \n

Padrões de nome de cidades do ZapImóveis: \n
    João Pessoa: pb+joao-pessoa \n
    Boa Vista: rr+boa-vista \n
    Maricá: rj+marica \n
    Recife: pe+recife \n
